# tmux

## 配置

``` bash
# curl
curl -o ~/.tmux.conf https://raw.githubusercontent.com/unihon/hon_conf/master/tmux/tmux.conf
# wget
wget -O ~/.tmux.conf https://raw.githubusercontent.com/unihon/hon_conf/master/tmux/tmux.conf
```
